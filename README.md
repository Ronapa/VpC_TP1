# VpC - TP1

Trabajo Práctico 1 de Visión por Computadora.

## Contenido

- **white_patch.ipynb**: corrección de color con el algoritmo White Patch
- **opencv_grayscale.ipynb**: visualización de imágenes en escala de grises e histogramas con OpenCV

## Requisitos

- Python 3.x
- Jupyter Notebook o JupyterLab

## Instalación

```bash
pip install -r requirements.txt
```

## Cómo correrlo

```bash
jupyter notebook
```
